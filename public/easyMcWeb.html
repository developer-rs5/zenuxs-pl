How It Works

1

Install the Plugin

Download and install easyMCWeb on your Minecraft server. It works with Spigot, PaperMC, and Bukkit.

2

Access Web Dashboard

Open the web interface through your browser. No port forwarding or complex setup required.

3

Design Your Website

Use the drag-and-drop builder to create pages, add elements, and customize your design.

4

Publish & Share

Click publish and share your website URL with players. Your website is now live!

Built-in APIs

// Example API Endpoints
GET /api/server/status
Response:
{
  "status": "online",
  "players": {
    "online": 42,
    "max": 100
  },
  "version": "1.20.1",
  "motd": "Welcome to our Server!"
}

GET /api/players/online
Response:
{
  "players": [
    {
      "name": "Player1",
      "uuid": "...",
      "playtime": 3600
    }
  ],
  "count": 42
}

GET /api/stats/server
Response:
{
  "uptime": 86400,
  "tps": 19.8,
  "memory": {
    "used": "4GB",
    "max": "8GB"
  }
}
